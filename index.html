2D Minecraft

This game is a project made by Jdd & JrM in one week for an assignment @ITC Coding Bootcamp.

Legend holds that he who finds the great axe has the power to smash trees and to starts a great adventure.


Minecraft is a sandbox game, destroy the blocks with the differents tools that are propose to you and create whatever you want with them !
Don't forget cloud and water can't be remove and you can only use the blocks that you already removed !!
